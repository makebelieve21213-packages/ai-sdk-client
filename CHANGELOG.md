# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-11-24

### Добавлено
- Базовая функциональность AI SDK клиента для NestJS
- Интеграция с OpenAI через Vercel AI SDK
- Streaming ответов через AsyncGenerator
- Управление контекстом разговора для контекстных ответов
- Поддержка Tools с автоматическим преобразованием JSON Schema в Zod схемы
- Context Data для передачи данных в tools
- NestJS модуль AiSdkModule с forRootAsync для гибкой конфигурации
- Гибкая конфигурация через переменные окружения через ConfigModule
- TypeScript типизация - полная типобезопасность
- 100% покрытие тестами

- Подробный README с примерами использования
- llms.txt для контекста ИИ агентов
- Инструкции по развертыванию в Docker
