# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.1] - 2025-11-25

### Исправлено
- Достигнуто 100% покрытие тестами по всем метрикам (Statements, Branches, Functions, Lines)
- Добавлены тесты для непокрытых веток выполнения с пустым `contextData` (`{}`)
- Улучшена обработка edge cases в тестах

### Изменено
- Использование `jsonSchema()` из AI SDK напрямую вместо внутренней утилиты преобразования
- Удалена утилита `json-schema-to-zod.ts` (больше не требуется)
- Удалена зависимость `zod` из package.json
- Обновлена документация с актуальной информацией о покрытии тестами

### Добавлено
- Экспорт `MessageType` enum в `index.ts` для использования в приложениях
- Детальная статистика покрытия тестами в README и llms.txt
- Тесты для всех веток выполнения с различными типами `contextData`

### Документация
- Обновлен README с информацией о 100% покрытии тестами
- Обновлен llms.txt с детальной статистикой тестов
- Исправлена структура пакета в документации

## [1.0.0] - 2025-11-24

### Добавлено
- Базовая функциональность AI SDK клиента для NestJS
- Интеграция с OpenAI через Vercel AI SDK
- Streaming ответов через AsyncGenerator
- Управление контекстом разговора для контекстных ответов
- Поддержка Tools с автоматической обработкой JSON Schema через AI SDK
- Context Data для передачи данных в tools
- NestJS модуль AiSdkModule с forRootAsync для гибкой конфигурации
- Гибкая конфигурация через переменные окружения через ConfigModule
- TypeScript типизация - полная типобезопасность
- 100% покрытие тестами

### Документация
- Подробный README с примерами использования
- llms.txt для контекста ИИ агентов
- Инструкции по развертыванию в Docker
- Руководство по внесению вклада (CONTRIBUTING.md)
